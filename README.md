# Sistema de Restituição de Bagagem - ELE634

Projeto desenvolvido para a disciplina de Laboratório de Sistemas II (UFMG) com o objetivo de automatizar o registro de chegada e saída de bagagens no Aeroporto de Confins.

## 🛠️ Tecnologias Escolhidas
- **Linguagem:** Python 3.x
- **Visão Computacional:** YOLOv8 (Detecção) + EasyOCR/pyzbar (Leitura IATA 740)
- **Banco de Dados:** SQLite
- **Hardware Sugerido:** Câmera de notebook / Webcam
